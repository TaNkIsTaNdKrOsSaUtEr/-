### **Конспект лекции №21: Галогенпроизводные углеводородов. Часть 1**

#### **1. Классификация и номенклатура**

**Галогенпроизводные углеводородов** – соединения, в которых один или несколько атомов водорода замещены на атом(ы) галогена.

**1.1. Классификация по типу гибридизации углеродного атома, связанного с галогеном:**

*   **Алкилгалогениды** (атом галогена связан с *sp³*-гибридизованным атомом C):
    *   **Примеры:**
        *   H₃C-CH₂-Cl (хлорэтан, этилхлорид)
        *   H₂C=CH-CH₂-Br (3-бромпропен, аллилбромид)
        *   C₆H₅-CH₂-Br (бромметилбензол, бензилбромид)

*   **Алкенилгалогениды (винилгалогениды)** и **Арилгалогениды** (атом галогена связан с *sp²*-гибридизованным атомом C):
    *   **Примеры:**
        *   H₂C=CH-Cl (хлорэтен, винилхлорид)
        *   C₆H₅-Cl (хлорбензол)

**1.2. Классификация дигалогенпроизводных по взаимному расположению атомов галогена:**

*   **Геминальные** (оба атома галогена у одного атома углерода): 1,1-дибромпропан
*   **Вицинальные** (атомы галогена у соседних атомов углерода): 1,2-дихлорпропан

#### **2. Способы получения**

**2.1. Из алканов: Галогенирование алканов**
*   **Механизм:** Свободнорадикальное замещение (Sᵣ).
*   **Реакция:** RH + Hal₂ → R-Hal + H-Hal
*   **Особенности:** Для хлора — низкая селективность, для брома — более высокая селективность (преимущественно атака на третичный атом C).

**2.2. Из алкенов:**

*   **2.2.1. Аллильное галогенирование**
    *   **Условия:** Высокая температура (400-500°C), свет (hν).
    *   **Механизм:** Свободнорадикальный (Sᵣ).
    *   **Пример:** Пропен + Cl₂ → 3-хлорпропен (аллилхлорид).

*   **2.2.2. Гидрогалогенирование**
    *   **Механизм:** Электрофильное присоединение (Adₑ).
    *   **Региоселективность:** Подчиняется правилу Марковникова.
    *   **Исключение:** Присоединение HBr в присутствии пероксидов идет против правила Марковникова (Adᵣ).

*   **2.2.3. Галогенирование**
    *   **Механизм:** Электрофильное присоединение (Adₑ).
    *   **Пример:** Пропен + Br₂ → 1,2-дибромпропан.

*   **2.2.4. Сопряженное присоединение**
    *   Присоединение в присутствии нуклеофильного реагента (H₂O, KCl).
    *   **Пример:** Пропен + Cl₂ + H₂O → 1-хлор-2-пропанол.

**2.3. Из диенов:**
*   **Реакции электрофильного присоединения** (1,2- и 1,4-присоединение).
*   **Пример:** 1,3-бутадиен + HBr → 3-бром-1-бутен (1,2-) и 1-бром-2-бутен (1,4-).

**2.4. Из аренов:**

*   **2.4.1. Электрофильное замещение в ядро**
    *   **Условия:** Катализаторы Льюиса (AlCl₃, FeBr₃).
    *   **Пример:** Толуол + Cl₂ → орто- и пара-хлортолуол.

*   **2.4.2. Свободнорадикальное замещение в боковую цепь**
    *   **Условия:** Высокая температура или свет (hν).
    *   **Пример:** Толуол + Cl₂ (hv) → бензилхлорид.

**2.5. Из спиртов:**
*   Замещение OH-группы на галоген.
*   **Реагенты:**
    *   **Для Cl:** HCl (часто с ZnCl₂), PCl₃, PCl₅, SOCl₂ (хлористый тионил).
    *   **Для Br:** HBr, PBr₃.
    *   **Для I:** HI, KI + H₃PO₄, P + I₂.

**2.6. Обмен галогенов:**
*   Реакция Финкельштейна: R-Cl(Br) + NaI → R-I + NaCl(Br)

**2.7. Из карбонильных соединений (альдегидов и кетонов):**
*   Действие пентагалогенидов фосфора (PCl₅, PBr₅).
*   **Пример:** CH₃-CH=O + PCl₅ → CH₃-CHCl₂

**2.8. Из солей диазония (для получения арилгалогенидов):**

*   **Реакция Шимана:** Разложение тетрафторбората арендиазония при нагревании.
*   **Реакция Зандмейера:** Разложение солей диазония в присутствии галогенидов меди (CuCl, CuBr).
