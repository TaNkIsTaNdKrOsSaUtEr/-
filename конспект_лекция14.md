### **Конспект лекции №14: Алкины**

#### **1. Общая характеристика и номенклатура**

*   **Общая формула:** \( C_nH_{2n-2} \)
*   **Гибридизация атомов углерода при тройной связи:** *sp*.
*   **Примеры названий:**
    *   HC≡CH — ацетилен (этин)
    *   H₃C-C≡CH — 1-бутин (пропилацетилен)
    *   H₃C-CH₂-C≡C-CH₂-CH(CH₃)₂ — 6-метил-3-гептин

#### **2. Способы получения**

**2.1. Карбидный метод**
*   Исходное сырье: CaCO₃.
    *   CaCO₃ →(t) CaO + CO₂
    *   CaO + 3C →(2000-2500 °C) CaC₂ + CO
    *   CaC₂ + 2H₂O → HC≡CH + Ca(OH)₂

**2.2. Пиролиз метана**
*   **Электрокрекинг:** 2CH₄ →(1500 °C) HC≡CH + 3H₂
*   **Термоокислительный крекинг:** 3CH₄ + O₂ →(1500 °C) HC≡CH + CO₂ + 5H₂

**2.3. Дегидрогалогенирование дигалогеналканов**
*   **2.3.1. Вицинальных дигалогеналканов** (атомы галогена у соседних атомов C).
    *   Действие спиртового раствора KOH (2 эквивалента).
    *   Протекает в две стадии: галогеналкан → алкен → алкин.
*   **2.3.2. Геминальных дигалогеналканов** (оба атома галогена у одного атома C).
    *   Также под действием сильных оснований.

**2.4. Дегалогенирование тетрагалогеналканов**
*   Обработка цинком при нагревании.
    *   Пример: R-CHCl-CCl₂-R' + 2Zn → R-C≡C-R' + 2ZnCl₂

**2.5. Алкилирование алкинов с концевой тройной связью**
*   **1-я стадия:** Получение ацетиленида.
    *   HC≡CH + NaNH₂ → HC≡C:⁻ Na⁺ + NH₃
    *   R-C≡CH + NaNH₂ → R-C≡C:⁻ Na⁺ + NH₃
*   **2-я стадия:** Реакция ацетиленида с алкилгалогенидом (R'-X).
    *   HC≡C:⁻ Na⁺ + R'-X → HC≡C-R' + NaX
    *   R-C≡C:⁻ Na⁺ + R'-X → R-C≡C-R' + NaX

#### **3. Химические свойства**

**3.1. Реакции электрофильного присоединения (Adₑ)**

*   **3.1.1. Галогенирование**
    *   Механизм: образование π-комплекса, затем бромониевого иона.
    *   Присоединение двух молекул галогена с образованием тетрагалогеналкана.
    *   Пример: R-C≡C-R + 2Br₂ → R-CBr₂-CBr₂-R

*   **3.1.2. Присоединение галогеноводородов**
    *   Присоединение подчиняется правилу Марковникова.
    *   Пример: R-C≡CH + 2HBr → R-CBr₂-CH₃

*   **3.1.3. Гидратация (реакция Кучерова)**
    *   Присоединение воды в присутствии солей ртути(II) (Hg²⁺).
    *   Конечный продукт — карбонильное соединение (кетон).
    *   Пример: R-C≡CH + H₂O →(Hg²⁺) R-C(O)-CH₃

*   **3.1.4. Присоединение карбоновых кислот**
*   **3.1.5. Присоединение спиртов**
    *   Образование виниловых эфиров.
    *   Пример: HC≡CH + CH₃OH →(CH₃ONa) CH₂=C(H)-OCH₃

**3.2. Реакции радикального присоединения (Adᵣ)**
*   Инициируются светом (hv) или пероксидами.
*   Пример: HC≡CH + HCl →(hv) H₂C=CH-Cl (хлористый винил)

**3.3. Реакции нуклеофильного присоединения**
*   Характерны для алкинов.
*   Пример: HC≡CH + C₂H₅OH →(C₂H₅ONa) CH₂=CH-OC₂H₅

**3.4. Реакции восстановления (гидрирования)**
*   **Полное гидрирование:** HC≡CH + 2H₂ →(Pt) CH₃-CH₃
*   **Частичное гидрирование до цис-алкена:** HC≡CH + H₂ →(Lindlar: Pd/CaCO₃, PbO) *цис*-CH=CH
*   **Частичное гидрирование до транс-алкена:** HC≡CH + 2Na →(NH₃(ж)) *транс*-CH=CH

**3.5. Реакции окисления**
*   Жесткое окисление (KMnO₄, K₂Cr₂O₄) приводит к расщеплению тройной связи с образованием карбоновых кислот.

**3.6. Кислотные свойства**
*   Атом водорода у терминального алкина (R-C≡C-**H**) обладает слабой кислотностью (pKa ~25).
*   Реагирует с очень сильными основаниями (NaNH₂, органощелочные металлы) с образованием ацетиленидов.
    *   HC≡CH + NaNH₂ → HC≡C:⁻ Na⁺ + NH₃
    *   HC≡CH + C₂H₅MgBr → HC≡C:⁻ MgBr⁺ + C₂H₆
*   Ацетилениды тяжелых металлов (например, Cu, Ag) нерастворимы и взрывоопасны в сухом виде.

**3.7. Реакции с карбонильными соединениями**
*   Ацетилениды вступают в реакцию с альдегидами и кетонами с образованием алкиновых спиртов.
*   **Пример:** RC≡C:⁻ Na⁺ + R'C(O)R'' → RC≡C-C(OH)R'R''
